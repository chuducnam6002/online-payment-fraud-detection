## 📥 Data Acquisition & Setup

Since the dataset is too large for GitHub (~470MB), please follow these steps to set up the environment:

1. **Download the Dataset:** Go to [Kaggle - Synthetic Financial Datasets](https://www.kaggle.com/datasets/ealaxi/paysim1) and download the ZIP file.

2. **Prepare the Data Folder:**
   Extract the file and ensure your local project structure looks like this:

   ```text
   ML.20252.Project/
   ├── data/
   │   └── raw/
   │       └── payment_fraud.csv  <-- Put the .csv file here
   ├── notebooks/
   │   └── 01_eda.ipynb
   └── README.md
